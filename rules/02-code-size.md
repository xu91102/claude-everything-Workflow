# 代码规模约束 (CRITICAL)

## 硬性限制

| 约束项      | 限制           | 违反后果               |
| ----------- | -------------- | ---------------------- |
| 单文件行数  | **≤ 600 行**   | 必须拆分文件           |
| 单函数/方法 | **≤ 80 行**    | 必须提取子函数         |
| 函数参数    | **≤ 5 个**     | 必须封装为对象         |
| 嵌套层级    | **≤ 4 层**     | 必须提前返回或提取函数 |
| 单行长度    | **≤ 120 字符** | 必须换行               |

**超限处理**：文件过长→拆分 | 函数过长→提取 | 参数过多→封装对象 | 嵌套过深→早返回
